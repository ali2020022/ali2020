# ⚡ Killua Codespace - Free VLESS+xHTTP Proxy

راه‌اندازی سریع و رایگان پروکسی VLESS با پروتکل خفن xHTTP روی GitHub Codespaces تو کمتر از ۵ دقیقه! بدون نیاز به سرور و دنگ و فنگ. 😎



## 🚀 شروع سریع (Quick Start)

۱. اول از همه این ریپو رو **Fork** کن.
۲. روی دکمه سبز **Code** کلیک کن، برو تب **Codespaces** و **Create codespace on main** رو بزن.
۳. یکی دو دقیقه صبر کن تا کانتینر لود بشه.
۴. **مهم‌ترین مرحله:** برو تو تب **PORTS** (پایین صفحه)، پورت **443** رو پیدا کن، روش راست‌کلیک کن و **Visibility** رو بذار روی **Public**. 🌐
۵. لینکی که تو ترمینال بهت داده رو کپی کن و بنداز تو برنامه‌ت (v2rayNG, V2Box, Nekobox و...). تمام! 🎉


## 🛠 دستورات به درد بخور (Commands)

هر وقت تو ترمینال کداسپیس بودی، اینا ممکنه به کارت بیاد:

- **دریافت دوباره لینک:** اگه لینک رو گم کردی کافیه بزنی:
  ```bash
  show-link.sh
  ```
- **جلوگیری از خاموش شدن کداسپیس و دیدن لاگ‌ها:**
  ```bash
  tmux attach -t killua
  ```
  *(برای خروج از این صفحه بدون اینکه سرور قطع بشه: اول `Ctrl+B` بعد `D` رو بزن)*

## 🧼 تغییر آی‌پی (Clean IPs)

اگه آی‌پی پیش‌فرض (`94.130.50.12`) برات کند بود، تو لینکی که کپی کردی یکی از این آی‌پی‌ها رو جایگزین کن:
- `63.141.252.203` (معمولاً روی ایرانسل جوابه)
- `50.7.5.83` (معمولاً روی همراه اول جوابه)

---

## ⚠️ دو تا نکته طلایی

- ⏳ **محدودیت رایگان:** گیت‌هاب ماهی **۶۰ ساعت** کداسپیس رایگان می‌ده. کارت که تموم شد حتماً کداسپیس رو استاپ (Stop) کن که تایمت الکی نسوزه.
- 🔒 **تنظیمات کلاینت:** اگه دستی تنظیم می‌کنی، حتماً `allowInsecure` فعال باشه (1) و `type` هم روی `xhttp` باشه.
